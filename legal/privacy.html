안전 개인정보 이용 기준

Local-first privacy

개인정보 안내

9회말 연구소는 로그인, 서버 계정, 실명 입력 없이 동작하는 정적 웹앱입니다. 현재 MVP는 카드 생성과 저장을 사용자 브라우저 안에서 처리합니다.

수집하지 않는 정보

  • 실명, 연락처, 이메일, 생년월일, 위치 정보
  • 실존 선수, 구단, 리그에 대한 개인 식별 목적의 정보
  • 로그인 계정, 결제 정보, 광고 식별자

브라우저에 남는 정보

보관함 카드, 팬심 반응 선택, 덕아웃 출석 기록은 이 브라우저의 로컬 저장소에만 남습니다. 사용자는 메인 화면의 내 데이터 영역에서 해당 기록을 지울 수 있습니다.

공유 링크와 PNG

공유 링크는 카드 문구와 선택 상태를 URL 해시에 담습니다. 서버로 전송하는 방식이 아니라 사용자가 직접 복사하거나 공유 시트를 통해 전달합니다. 다만 링크를 공유하면 채팅 앱, 브라우저 기록, 링크를 받은 사람에게 카드 문구가 보일 수 있습니다. PNG 저장은 브라우저 안에서 생성됩니다.

서비스 워커 캐시

홈 화면 추가와 빠른 재방문을 위해 HTML, CSS, JavaScript, 이미지 같은 정적 파일을 브라우저 캐시에 저장할 수 있습니다. 캐시에는 로그인 정보나 실명 정보가 포함되지 않습니다.

권리·안전 검토 요청

현재 검토 요청 기능은 서버 접수가 아니라 운영자에게 보낼 문구를 만들어 복사하는 기능입니다. 입력한 메모는 외부로 자동 전송되지 않습니다. 공개 배포 전에는 연락, 삭제 요청, 처리 기준을 별도로 마련해야 합니다.

이 안내는 2026년 6월 15일 기준 MVP 동작을 설명합니다. 향후 서버, 계정, 광고, 중개 기능이 추가되면 별도 고지와 동의 흐름이 필요합니다.
